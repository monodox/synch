# Roadmap

## v1.0 — Core foundation (current)

- [x] Execution pipeline with computation reuse
- [x] In-memory cache with TTL and size limits
- [x] State management for execution tracking
- [x] Concurrency control with timeouts
- [x] Request routing and validation
- [x] CLI mode (inline, file, stdin)
- [x] Web server mode (HTTP API)
- [x] TypeScript type definitions
- [x] Test suite

## v1.1 — Enhanced caching

- [ ] LRU and LFU eviction strategies
- [ ] Cache persistence to disk
- [ ] Redis adapter for distributed caching
- [ ] Custom cache key generation
- [ ] Cache warming and preloading

## v1.2 — Reliability

- [ ] Request batching and deduplication
- [ ] Retry logic with exponential backoff
- [ ] Circuit breaker pattern
- [ ] Structured logging
- [ ] Metrics and monitoring hooks

## v2.0 — Extensibility

- [ ] Plugin system for custom extensions
- [ ] Middleware pipeline
- [ ] Custom state storage adapters (Redis, SQLite)
- [ ] Streaming support for large outputs
- [ ] AI model adapters (OpenAI, Anthropic, etc.)

## v2.1 — Intelligence

- [ ] Semantic caching for similar inputs
- [ ] Smart routing based on input patterns
- [ ] Cost and token usage tracking
- [ ] Dynamic concurrency adjustment
- [ ] Predictive cache preloading

## v3.0 — Scale

- [ ] Distributed execution across nodes
- [ ] Workload balancing and scheduling
- [ ] Multi-tenant support
- [ ] Observability and tracing
- [ ] Web dashboard
